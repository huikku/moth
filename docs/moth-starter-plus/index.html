MOTH Minimal Overhead Technical Hierarchy
Tiny specs for big brains.

MOTH is a compact, human-readable notation for PRDs, rules, and architecture that AI coding agents can load always-on without hogging tokens. Sister project to kablUI.

Always-in-context

        
Load MOTH files first in Cursor/Cline/Aider so agents share a single source of truth.

        

          
Token profile
~5–10× smaller than prose

          
Format
sectioned [TAGS], key:value, semicolon lists

          
Use
.moth files: PRD, rules, architecture

        

      

      

        
LLM-first design

        
Low-entropy tokens, deterministic structure, easy to parse with regex or PEG.

        

          
Reserved
[CORE], [STACK], [API.*], [SCHEMAS.*]

          
Validation
intellimoth validate (CLI)

          
CLI
compile | validate | stats | expand
